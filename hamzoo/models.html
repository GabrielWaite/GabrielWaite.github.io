The Heisenberg Model

$H=\displaystyle\sum_{i,j\in\Lambda}J_{ij}\big(X_iX_j + Y_iY_j + Z_iZ_j \big)$

The Fermi-Hubbard Model:

$H=\displaystyle\sum_{i,j\in\Lambda}\sum_{\sigma}t_{ij} c_{i\sigma}^\dagger c_{j\sigma} + \displaystyle\sum_{i\in\Lambda}U_{ij}n_{i\uparrow}n_{i\downarrow}$

The Bose-Hubbard Model:

$H=\displaystyle\sum_{i,j\in\Lambda}t_{ij} a_i^\dagger a_j + \frac{U}{2}\displaystyle\sum_{i\in\Lambda}n_i(n_i - 1)$

The Transverse Ising Model:

$H=\displaystyle\sum_{i,j\in\Lambda}L_{ij}Z_iZ_j + \displaystyle\sum_{i\in\Lambda}f_iX_i$

$2$-local Pauli Models:

$H=\displaystyle\sum_{\mu,\nu\in\Lambda} \lambda_{\mu\nu}P_\mu \otimes Q_\nu$

The Parent Pauli Hamiltonian:

$\displaystyle\sum_{i,j\in\Lambda}J_{ij}X_iX_j + K_{ij}Y_iY_j + L_{ij}Z_iZ_j + \displaystyle+ \sum_{i,j\in\Lambda}P_{ij}X_iY_j + Q_{ij}X_iZ_j + R_{ij}Y_iX_j + S_{ij}Y_iZ_j + T_{ij}Z_iX_j + V_{ij}Z_iY_j + \displaystyle+ \sum_{i\in\Lambda} f_iX_i + g_iY_i + h_iZ_i$

The $\textsc{x-y-z/.}$ Model:

$H=\displaystyle\sum_{i,j\in\Lambda}J_{ij}X_iX_j + K_{ij}Y_iY_j + L_{ij}Z_iZ_j$

The $\textsc{x-y/.}$ Model:

$H=\displaystyle\sum_{i,j\in\Lambda}J_{ij}X_iX_j + K_{ij}Y_iY_j$

The $\textsc{xy/.}$ Model:

$H=\displaystyle\sum_{i,j\in\Lambda}J_{ij} \big(X_iX_j + Y_iY_j\big)$

The $\textsc{xy-z/.}$ Model:

$H=\displaystyle\sum_{i,j\in\Lambda}J_{ij}\big(X_iX_j + Y_iY_j\big) + L_{ij}Z_iZ_j$

The $\textsc{x-z/x-z}$ Model:

$H=\displaystyle\sum_{i,j\in\Lambda} J_{ij} X_iX_j + L_{ij} Z_iZ_j + \sum_{i\in\Lambda} f_i X_i + h_i Z_i$

Gross name model:

$H=\displaystyle\sum_{i,j\in\Lambda} Q_{ij} X_iZ_j + T_{ij} Z_iX_j + \sum_{i\in\Lambda} f_i X_i + h_i Z_i$

Weighted $\textsc{xyz/.}$ Model:

$H=\displaystyle\sum_{i,j\in\Lambda} J_{ij} \big( \alpha X_iX_j + \beta Y_iY_j + \gamma Z_iZ_j\big)$

Electronic Structure Model:

$H=\displaystyle\sum_{i,j \in \Lambda}\sum_{\sigma} t_{ij} c_{i\sigma}^\dagger c_{j\sigma} + \frac{1}{2} \displaystyle\sum_{i,j,k,l \in \Lambda}\displaystyle\sum_{\sigma,\tau} u_{ijkl}c_{i\sigma}^\dagger c_{j\tau}^\dagger c_{k\sigma} c_{l\sigma}$

The $t$-$J$ Model*:

$H=\displaystyle\sum_{i,j\in\Lambda}\frac{4t_{ij}^2}{U} \boldsymbol{S}_i \cdot \boldsymbol{S}_j$
