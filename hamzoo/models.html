Here we categorise many of the common local Hamiltonian models encountered in literature. For example, the Heisenberg and Fermi-Hubbard models are frequently referenced in research. This page aims at providing general definitions for such. See supplimentary material for strict definitions on what it means to have a local Hamiltonian.

We often define some Hamiltonian based on its interactions. Many of those shown below are $2$-local interactions meaning the underlying interaction graph, denoted $\Lambda$ directly describes the interactions that occur. Other common notation for the underlying graph stems from the graph theoretical framework where we can instead represent $\Lambda$ as some graph $G(E,V)$ with an edge set, $E(G)$ and a vertex set $V(G)$. The $2$-local interactions just mentioned then take place over the edges of this graph. Moreover, there exists and edge $e\in E(G)$ of weight $J_{i,j}$, if there exists an interaction between the two vertices, $i,j \in V(G)$. Hence the graph strucutre completely specifies the Hamiltonian. If, for example, the Hamiltonian presents as,

$H=\displaystyle\sum_{i,j\in \Lambda} J_{ij}\; X_iX_j + K_{ij}\;Y_iY_j$

Then between some verticies $i,j$, there can be multiple edges of weight $J_{ij}$ and $K_{ij}$. In fact, this is true regardless of the 2 vertices, however it could be that for some vertex $k$ and another vertex $l$, $J_{kl}=0$ and hence no edge exists!

Conventions

Within many-body physics there exists a naming convention for the different kinds of models that can arise. For example, the XXZ model hosts a Hamiltonian of the form,

$H=\displaystyle\sum_{i,j\in \Lambda} J_{ij}\big(X_iX_j + Y_iY_j\big) + L_{ij}\;Z_iZ_j$

Other such examples exist however a universal reference to such models can sometimes be difficult and often breaksdown. Within some literature there is a conflict between what the XY model is. The two possibilites can be,

$H_1=\displaystyle\sum_{i,j\in \Lambda} J_{ij}\big(X_iX_j + Y_iY_j\big)$, $H_2=\displaystyle\sum_{i,j\in \Lambda} J_{ij}\;X_iX_j + K_{ij}\;Y_iY_j$

There is a subtle and important difference between these two models, especially when it comes to complexity theory. It is trivial to see that $H_1$ is a special case of $H_2$ when $K_{ij}=J_{ik}$. Due to this confusion a new naming convention was proposed in (maybe the review we write), in hopes of generating a universal naming convention that completely specifies the important aspects of the model. Within this novel naming convention the difference between $H_1$ and $H_2$ can be seen via,

$H_1 \implies$(xy/.) Hamiltonian, $H_2 \implies$(x-y/.) Hamiltonian

The slight difference between the two models can trivially be seen using this convention. Moreover, the fact that $H_1$ is a special case of $H_2$ is also clear.

The Heisenberg Model

$H=\displaystyle\sum_{i,j\in\Lambda}J_{ij}\big(X_iX_j + Y_iY_j + Z_iZ_j \big)$

The Fermi-Hubbard Model:

$H=\displaystyle\sum_{i,j\in\Lambda}\sum_{\sigma}t_{ij} c_{i\sigma}^\dagger c_{j\sigma} + \displaystyle\sum_{i\in\Lambda}U_{ij}n_{i\uparrow}n_{i\downarrow}$

The Bose-Hubbard Model:

$H=\displaystyle\sum_{i,j\in\Lambda}t_{ij} a_i^\dagger a_j + \frac{U}{2}\displaystyle\sum_{i\in\Lambda}n_i(n_i - 1)$

The Transverse Ising Model:

$H=\displaystyle\sum_{i,j\in\Lambda}L_{ij}Z_iZ_j + \displaystyle\sum_{i\in\Lambda}f_iX_i$

$2$-local Pauli Models:

$H=\displaystyle\sum_{\mu,\nu\in\Lambda} \lambda_{\mu\nu}P_\mu \otimes Q_\nu$

The Parent Pauli Hamiltonian:

$\displaystyle\sum_{i,j\in\Lambda}J_{ij}X_iX_j + K_{ij}Y_iY_j + L_{ij}Z_iZ_j + \displaystyle+ \sum_{i,j\in\Lambda}P_{ij}X_iY_j + Q_{ij}X_iZ_j + R_{ij}Y_iX_j + S_{ij}Y_iZ_j + T_{ij}Z_iX_j + V_{ij}Z_iY_j + \displaystyle+ \sum_{i\in\Lambda} f_iX_i + g_iY_i + h_iZ_i$

The (x-y-z/.) Model:

$H=\displaystyle\sum_{i,j\in\Lambda}J_{ij}X_iX_j + K_{ij}Y_iY_j + L_{ij}Z_iZ_j$

The (x-y/.) Model:

$H=\displaystyle\sum_{i,j\in\Lambda}J_{ij}X_iX_j + K_{ij}Y_iY_j$

The (xy/.) Model:

$H=\displaystyle\sum_{i,j\in\Lambda}J_{ij} \big(X_iX_j + Y_iY_j\big)$

The (xy-z/.) Model:

$H=\displaystyle\sum_{i,j\in\Lambda}J_{ij}\big(X_iX_j + Y_iY_j\big) + L_{ij}Z_iZ_j$

The (x-z/x-z) Model:

$H=\displaystyle\sum_{i,j\in\Lambda} J_{ij} X_iX_j + L_{ij} Z_iZ_j + \sum_{i\in\Lambda} f_i X_i + h_i Z_i$

Gross name model:

$H=\displaystyle\sum_{i,j\in\Lambda} Q_{ij} X_iZ_j + T_{ij} Z_iX_j + \sum_{i\in\Lambda} f_i X_i + h_i Z_i$

Weighted $\textsc{xyz/.}$ Model:

$H=\displaystyle\sum_{i,j\in\Lambda} J_{ij} \big( \alpha X_iX_j + \beta Y_iY_j + \gamma Z_iZ_j\big)$

Electronic Structure Model:

$H=\displaystyle\sum_{i,j \in \Lambda}\sum_{\sigma} t_{ij} c_{i\sigma}^\dagger c_{j\sigma} + \frac{1}{2} \displaystyle\sum_{i,j,k,l \in \Lambda}\displaystyle\sum_{\sigma,\tau} u_{ijkl}c_{i\sigma}^\dagger c_{j\tau}^\dagger c_{k\sigma} c_{l\sigma}$

The $t$-$J$ Model*:

$H=\displaystyle\sum_{i,j\in\Lambda}\frac{4t_{ij}^2}{U} \boldsymbol{S}_i \cdot \boldsymbol{S}_j$
