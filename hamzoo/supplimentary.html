Terminology

Quantum Computer: A quantum computer is a computational machine that utilises the fundamental principles of quantum mechanics to execute computation tasks. The primitive elements of a quantum computer are quantum bits (qubits) â€” naturally exhibiting superposition and entanglement properties. There attributes generate the quantum advantage in information processing not efficiently produced on classical computers.

Merlin: Merlin (prover) exists as an omniscient, omnipotent and mendacious entity, having unbounded computational resources - essentially representing an infintely powerful oracle. Additionally, Merlin possess the ability to generate quantum states. The role of Merlin is to convince Arthur (verifer) as to the validity of a decision problem's solution via some proof/witness state.

Arthur:Arthur (verifer) is a computational entity, typically represented as a Turing Machince, thus having bounded capabilities. Arthur's job is verify the solutions for a given decision problem using the proof/witness state provided by Merlin (prover). Based on certain criterion, Arthur will accept or reject with some probability.

Polynomial Time: In computational theory, a polynomial time algorithm is an algorithm whose running time is bounded by a polynomial function of the input size. More formally, an algorithm is said to run in polynomial time if there exists a polynomial function $f(n)$, where $n$ represents the size of the input, such that the algorithm's running time is $O\big(f(n)\big)$.
